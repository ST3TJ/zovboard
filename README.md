# zovboard

Простой Windows-хук для переназначения русских букв на латиницу.

## Особенности
- Переназначает `з → Z`, `в → V`, `о → O`.
- Работает через глобальный низкоуровневый хук клавиатуры.
- Автоматически отправляет заменённые символы в активное окно.

## Сборка
Компилировать под Windows x64 с поддержкой C++14 или выше:
```bash
cl /EHsc /W4 /MD zovboard.cpp
```

## Использование
1. Программа будет работать в фоне, перехватывая нажатия клавиш.
2. Для выхода закрыть окно консоли.

## Примечания
- Подходит только для Windows.
