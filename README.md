# zovboard

Windows-хук для автоматической замены русских букв на латиницу (Z, O, V) с работой в системном трее.

## Особенности
- **Ремаппинг:** Заменяет `з → Z`, `в → V`, `о → O`.
- **System Tray:** Работает в фоне, сворачивается в область уведомлений (трей).
- **Context Menu:** Управление через контекстное меню иконки.
- **Stealth:** Не создает окна консоли, работает полностью скрытно.

## Сборка
Так как программа переписана под оконный интерфейс (WinAPI) и использует ресурсы (иконку), для сборки требуются флаги подсистемы Windows и линковка системных библиотек.

Требования: MSVC (Visual Studio Build Tools), Windows SDK.

```cmd
:: 1. Компиляция файла ресурсов (предполагается наличие resource.rc)
rc resource.rc

:: 2. Компиляция и линковка
:: /SUBSYSTEM:WINDOWS — убирает окно консоли при запуске
cl /nologo /W4 /EHsc /DUNICODE /D_UNICODE zovboard.cpp resource.res user32.lib shell32.lib /link /SUBSYSTEM:WINDOWS
```
